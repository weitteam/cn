更新日志

布斯起始页全新重构，我们主动抛弃长期以来的开发成果，选择从头开始

本次更新：

  • 增加个人设置导入导出功能，数据丢失是不存在的啦！
  • 增加夜间模式风格，只要你的电脑开启了就自动应用啦！
  • 自定义背景设置项，增加 Unsplash 随机图片源
  • 增加导航项拖拽排序功能
  • 增加导航项管理抽屉，可自由添加和删除你想要的导航项
  • 增加运行模式判断，用户模式下（地址栏输入 ?u=paul）仅可读，无法使用任意编辑功能
  • CSS 样式和细节优化，JS 代码逻辑优化
  • 优化关闭弹窗和抽屉的动画和速度
  • 解决使用抽屉添加导航项导致无法拖拽的问题
  • 打开起始页时，可自动聚焦到搜索框
  • 修复了搜索下拉框不会收回去的问题
上次更新：
  • 优化代码命名和逻辑，增加可读性，更新为现代 JavaScript 写法，鼓励你和我一起参与本项目的 维护！
  • 窗口关闭按钮改为事件委托，修复用户模式下的 Bug（无法进行设置）
  • 增加了搜狗搜索
  • 替换了新的橙色 Logo
  • 调整导航项 DOM 结构，修改版权展示时间
  • 解决了在用户模式下（地址栏输入 ?u=paul）配置会被本地覆盖的 Bug
  • 在用户模式下，禁止使用设置功能
  • 页脚版权的时间自动生成，过期（气）是不存在的啦！
更新预告：
  • 更舒适的自定义站点列表体验
  • 可供设定你自己的壁纸（本地或外链）
